# locustjs-base
This library contains basic functions.

Current basic functions are as follows:

* isString      
* isNumber      
* isDate        
* isBool        
* isaN      	
* isEmpty       
* isSomeString  
* isObject      
* isSomeObject  
* isFunction    
* isNumeric     
* hasDate       
* hasBool		
* isFormatedDate
* isPrimitive   
* isArray		
* isSomeArray	
* isNamespace

Also, the library provides two BaseEnum and Enum utility classes to define enums.
